# ADR_BINNs

This is the official repository for the paper: **"Reliable mechanistic operator recovery with biologically-informed neural networks: principles for architecture and optimisation design."**

## Repository Contents

This project includes:

* **Training Scripts (`.py`):** Python files to train Biologically-Informed Neural Networks (BINNs) on various Partial Differential Equations (PDEs). We use these to test different network setups and hyperparameter choices.
* **Plotting Notebooks (`.ipynb`):** Jupyter notebooks to reproduce the figures used in the manuscript. Please note that, although plotting the same underlying data and results, these notebooks do not contain the most up-to-date formatting choices.
* **Generated Figures (`.zip`):** Compressed folders containing the final produced figures.

## Data Access

The BINN output data used to generate the plots is publicly available on Hugging Face: [https://huggingface.co/datasets/IamYuanYinOxford/ADR_BINNs_data](https://huggingface.co/datasets/IamYuanYinOxford/ADR_BINNs_data)
