# ADR_BINNs

This is the official repository for the paper: **"On the sensitivity of biologically-informed neural networks to architecture and hyperparameter design choices."**

## Repository Contents

This project includes:

* **Training Scripts (`.py`):** Python files to train Biologically-Informed Neural Networks (BINNs) on various Partial Differential Equations (PDEs). We use these to test different network setups and hyperparameter choices.
* **Plotting Notebooks (`.ipynb`):** Jupyter notebooks to reproduce the figures used in the manuscript.
* **Generated Figures (`.zip`):** Compressed folders containing the final produced figures.
